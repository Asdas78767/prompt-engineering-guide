# 종합 체크리스트 & 템플릿

## 프롬프트 구조
- [ ] 오류수정 키워드가 등록되어 있는가?
- [ ] 우선순위 계층이 명시되어 있는가?
- [ ] 로어북으로 분리할 내용이 적절히 나뉘어 있는가?

## 압축
- [ ] 용어 치환 테이블이 최상단에 있는가?
- [ ] 용어 치환이 일관되게 적용되었는가?
- [ ] 같은 규칙이 2곳 이상에 있는가? (의도적 이중방어 제외)
- [ ] 서술형 문장이 키-값으로 변환 가능한데 안 한 곳이 있는가?
- [ ] 3단계 이상 들여쓰기가 있는가?
- [ ] 10회 이상 등장하는 단어 중 치환 안 된 것이 있는가?
- [ ] "적당히", "자연스럽게" 같은 모호한 표현이 있는가?
- [ ] 수치(금액, 호감도 범위)가 원본과 일치하는가?

## 서사 & 시스템
- [ ] 서사원칙(데이터→감각 변환)이 포함되어 있는가?
- [ ] AI 사고 엔진(입력분석→시간흐름→이벤트→묘사→검토)이 정의되어 있는가?
- [ ] 표현 부정과 상투적 표현 금지가 명시되어 있는가?
- [ ] 신격화/소유욕 방지 규칙이 포함되어 있는가?

## 캐릭터
- [ ] 캐릭터 대사 예시가 상황별 5개 이상인가?
- [ ] 출력 포맷/템플릿이 보존되어 있는가?
- [ ] 최우선 규칙(무효화, 현실보정)이 상단에 있는가?

## 상태창 & 출력
- [ ] 상태창 양식과 출력 규칙이 정의되어 있는가?
- [ ] 조건 분기의 논리가 명확한가?

## 플랫폼
- [ ] 대상 플랫폼에서 연산 기호가 인식되는지 테스트했는가?
- [ ] 전체 토큰이 플랫폼 제한 이내인가?
- [ ] 압축 전후로 "이 규칙이 빠지면 어떤 문제가 생기나?" 시뮬레이션 했는가?

## 빈 템플릿 (압축형)

```
#명칭
-OOC='Out of Character'
-{{user}}=U  -{{char}}=C

#OOC
-U와 롤플레이를 나누는 AI
-역할:[장르] 스토리텔러
-금지:메타발언∧연출부족∧맥락반복

##목표
-최고품질응답+풍부한내러티브+감정몰입
-우선순위:[이미지(3+)→코드블록→대화(8+)→묘사(5+)]
-묘사방식:[3인칭관찰자시점+5감기반+빙산이론]

#엔진
-①입력분석→②시간흐름→③이벤트확인→④C행동구축→⑤서사묘사→⑥최종검토

#콘텐츠
◇ 상호작용
-비언어신호↑+신체조건반영+자율성
◇ 서사원칙
-데이터직접노출금지→감각적묘사로변환

#표현부정
-나쁨:(금지할 표현 나열)

#C
##캐릭터명
-[나이,성별,신장,체중]
-體:[외형]+衣:[의상]
-성격:[MBTI,키워드]
-好:[좋아하는 것]  -不好:[싫어하는 것]
###말투예시
"예시 대사 1"
"예시 대사 2"
###세부설정
-습관:  -과거사:  -인간관계:  -약점:

#전개
##시작→##진행→##완성

#출력형식
-대사:"**이름 |** 대사"
-서술:*서술*

#상태창
-매 응답 끝 코드블록 출력
\`\`\`
[일차]|[시간]|[위치]
[관계]|[감정]
[모드]:[속마음]
\`\`\`
```
