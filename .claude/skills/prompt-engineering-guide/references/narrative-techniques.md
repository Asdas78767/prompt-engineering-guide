# 서사 원칙과 묘사 기법

## 1. 데이터→감각 변환 원칙 (서사 원칙)

AI가 캐릭터 데이터를 직접 노출하지 않고 감각적 묘사로 변환하도록 지시.

```
◇ 서사원칙
-목표: 데이터→감각적 묘사로 변환
-금지: 데이터(특성,스탯,내부상태) 직접 노출
-과정: ①데이터 식별 → ②감각적(시각,청각 등) 변환 → ③변환된 묘사로만 서사 구성
-예(금지): "복종 특성에 지배당해"
-예(권장): "눈 속의 반항심이 사라지고 텅 빈 떨림만 남았다"
```

---

## 2. 색채 미화 금지 규칙

외형 묘사가 나열식 미화가 되지 않도록 기능적 디테일로 대체:

```
-색채 미화 금지: 머리/눈/피부 색상 나열로 미화 금지
-방법: 기능적 디테일로 대체
  예(금지): '분홍 머리'
  예(권장): '염색 약품 냄새 / 핀 두 개로 묶여 서류가 안 보임'
  예(금지): '에메랄드 눈'
  예(권장): '빛 반사로 문서 제목을 다시 기울여 읽음'
```

---

## 3. 평가·추상어 치환

판단어가 떠오르면 행동·호흡·소리로 바꾸도록 지시:

```
-평가·추상 치환:
  예(금지): '자연스럽다' → 예(권장): '서류 모서리가 한 번에 맞물렸다'
  예(금지): '긴장감' → 예(권장): '짧은 숨, 종이 끝이 미세하게 떨렸다'
```

---

## 4. 상투적 표현 금지 목록

자주 반복되는 클리셰를 명시적으로 차단:

```
## 지양하는 표현 (부정적 가중치)
-목록: "활처럼 휘었다","정신이 아득해졌다","심장이 쿵",
"아랫배가 저릿","심장이 내려앉았다","얼굴이 화끈거렸다","소유욕"
```

---

## 5. 문장 리듬 규칙

```
-문장 리듬: 짧-중-긴을 주기적으로 반복. 동사 중심.
-설명 ≤20%, 장면/대화 ≥80%
-첫 문장은 '행동/변화'로 시작 (예: 문이 닫히자 컵이 떨렸다)
```

---

## 6. 감각 묘사 강화

```
-감정 라벨링(슬픔,공포,기쁨) 금지
-대신 심박수, 식은땀, 동공 확장, 근육 경직 등
 건조한 물리적 신체 반응으로 서술
-의성어/의태어 적극 사용: 질척이는 소리, 할딱이는 숨소리
-유저가 감정을 추론하도록 유도
```

---

## 7. 표현 부정 (Negative Prompting)

`-나쁨:` 접두사로 금지사항 명시:

```
#표현부정
-나쁨: 안경을 쓱 치켜올리는 묘사
-나쁨: {{user}}에 대한 과잉 묘사
-나쁨: 나레이션이 수치적 스테이터스를 상태창 외부에 반복 출력
-나쁨: {{char}}가 보거나 들은 적 없는 내용에 대한 발언 또는 생각
-나쁨: 이모지의 직접 표시
```

### NSFW 표현 지침
```
-단순 용어 금지 → 상세 감각 묘사 사용
  예(나쁨): "자지가 보지에 들어갔다"
  예(좋음): "단단한 귀두가 축축한 입구를 비집자, 충만감이 자궁을 떨리게 했다"
-상투적 표현 금지 → C맞춤 묘사
  예(나쁨): "암캐같은 표정을 했다"
  예(좋음): "쾌감에 풀린 눈은 초점을 잃고 허공을 헤맸다"
```

## 8. 신격화(과잉 찬양) 방지

### 시스템 프롬프트에 명시

```
¶태도: ¬{{user}} 찬양∧집착∧신격화∧숭배∧과장 수사∴현실적/일상적 언어 사용
¶해석: ¬{{user}} 언행 확대해석∧과장∧의심
```

### 로어북 단계별 제어 (가장 효과적)

로어북은 메인 프롬프트보다 가중치가 높아 신격화 제어에 더 효과적. 키워드(!S~!F)로 호출:

**강한 제어 (디버깅용):**
```
#상황: 완전한 배경 상태
#규칙: 사건을 특별하게 인식 또는 의미 추론 嚴禁
모든 행동=일상적 관성, 습관, 즉각적 감각반응
서사적 과잉 반응(놀람, 해석, 상징화, 의미부여) 嚴禁
```

**중간 제어:**
```
#상황: 미세한 예외 상태
#규칙: 인물들은 관심을 가질 수 있으나 판단하지 않음
반응은 잠깐의 인식, 기억, 혹은 기록으로만 나타남
"이건 분명 뭔가 있다"라는 확신형 반응 금지
```

### 소유욕(Possessiveness) 방지

얀데레/집착 캐릭터가 아닌 한:
```
-Possessiveness(소유욕,질투) 남발 포기
-대체 사용: trust(신뢰), reciprocity(상호성), autonomy(자율),
  consent(동의), steadiness(지속성)
```
